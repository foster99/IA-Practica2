# IA-Practica2
Practica 2 de la asignatura IA-FIB sobre sistemas basados en el conocimiento (SBC). Recomendador de libros.

Instrucciones de uso:
	
	1 - Ejecutar la IDE de clips.
	
	2 - Cargar el fichero con la instruccion:
		> (load "<Ruta>/recomendador.clp")
		
	3 - Ejecutar la instruccion de reset para cargar las instancias:
		> (reset)
	
	4 - Ejecutar el comando de ejecutar para iniciar el programa:
		> (run)
